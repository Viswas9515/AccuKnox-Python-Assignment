# AccuKnox Python Assignment

## Problem Statements Implemented

### 1. System Health Monitoring Script
Checks:
- CPU Usage
- Memory Usage
- Disk Usage
- Running Processes

Displays alerts if CPU, Memory, or Disk usage exceeds 80%.

---

### 2. Application Health Checker

Checks whether a web application is UP or DOWN using HTTP status codes.

---

## Requirements

```bash
pip install -r requirements.txt
```

## Run System Health Script

```bash
python system_health.py
```

## Run Application Health Checker

```bash
python application_health.py
```
